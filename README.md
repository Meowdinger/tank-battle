# 坦克大战 Tank Battle

经典 FC 红白机版坦克大战的 HTML5 复刻版，使用纯 Canvas 渲染，无任何依赖，单文件可直接打开运行。

## 在线试玩

打开 `index.html` 即可游玩，或部署到 GitHub Pages 后通过仓库主页访问。

## 操作说明

| 按键 | 功能 |
| --- | --- |
| `←` `↑` `↓` `→` 或 `WASD` | 移动坦克 |
| `空格` | 开火 |
| `P` | 暂停 / 继续 |
| `Enter` | 开始游戏 / 返回菜单 |

## 游戏规则

- 玩家驾驶黄色坦克，消灭从地图顶部出现的所有敌方坦克即可通关
- 必须保护地图底部的基地（鹰巢）不被摧毁，否则游戏结束
- 玩家初始 3 条命，被击中且未持有星标时损失 1 条
- 每关共 20 辆敌方坦克，同屏最多 4 辆

## 敌方坦克种类

| 类型 | 颜色 | 特性 | 击杀得分 |
| --- | --- | --- | --- |
| 基础型 | 灰白 | 速度普通，子弹普通 | 100 |
| 快速型 | 浅蓝 | 移动迅捷 | 200 |
| 火力型 | 浅绿 | 子弹速度快 | 300 |
| 装甲型 | 灰/黄/绿/红 | 4 点护甲，颜色随血量变化 | 400 |

## 道具系统

击毁闪烁的敌方坦克可掉落随机道具：

- ★ 武器升级：累计 3 级后子弹更强，可穿钢墙、破墙更宽
- 盾牌：10 秒护盾，免疫所有伤害
- 炸弹：消灭场上所有敌人
- 红色 1UP：额外生命 +1
- 雪花：冻结敌人 8 秒

## 地形

- 砖墙：子弹可破坏
- 钢墙：普通子弹无效，需武器升至 2 星以上
- 水面：坦克无法通过，子弹可飞越
- 树丛：坦克通过时被掩盖
- 基地：被击中即游戏结束

## 项目结构

tank-battle/

## 技术细节

- 渲染：HTML5 Canvas 2D
- 帧率：60 FPS（requestAnimationFrame）
- 坐标：13×13 主格 + 26×26 子格（用于砖墙的精细破坏）
- 无外部依赖，无构建步骤

## 浏览器要求

支持 ES2015+ 与 Canvas 2D 的现代浏览器（Chrome / Firefox / Edge / Safari 任意近期版本）。
